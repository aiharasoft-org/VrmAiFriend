# VrmAiFriend

VRM キャラクターと Gemini Live でリアルタイム会話ができるデスクトップコンパニオンです。  
AI の応答は **Aivis Speech** で音声合成して再生します。会話からダンスを踊らせたり、カメラの構図を調整できます。

## インストール

起動前に、以下を用意してください。

| 項目                    | 内容                                                                                                                                                                    |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Aivis Speech Engine** | 音声合成に必須。PC にインストールし、起動しておく（既定: `http://127.0.0.1:10101`）。[AivisSpeech Engine](https://github.com/Aivis-Project/AivisSpeech-Engine) を参照。 |
| **Gemini API キー**     | `~/.env` に `GEMINI_API_KEY=...` を記載する。                                                                                                                           |

> **注意:** `GEMINI_API_KEY` が未設定、または無効な場合、Gemini Live に接続できず **会話できません**。

メニューの「音声」欄に **Aivis Speech 接続OK** と表示されれば、音声合成の準備はできています。  
**Aivis Speech を起動してください** と出る場合は、Aivis Speech Engine を起動してから「Aivis 一覧を更新」を押してください。

## 使い方

1. Aivis Speech Engine を起動する。
2. `~/.env` に `GEMINI_API_KEY` を設定する。
3. アプリを起動する。
4. マイクに向かって話す（起動時に Gemini Live へ自動接続）。
5. AI の応答が Aivis Speech で音声合成され、口パクとともに再生される。

### メニュー操作

`ESC` キーでメニューとウィンドウの透明表示を切り替えられる。

**VRoid Hub**

- **ハートしたモデルを表示** — ログインし、ハートしたモデル一覧を取得
- 一覧からモデルを選び、利用条件に同意後に VRM を読み込む
- **クレジット表記** — 現在の Hub モデルのクレジットを表示
- **VRoid Hubを開く** — ブラウザで VRoid Hub を開く

**音声（Aivis Speech）**

- **接続ステータス** — Aivis Speech の状態を表示
- **音声モデル** — サムネイル付き一覧から話者を選択（初回は「コハク」）
- **スタイル** — ノーマルなどの話し方を選択
- **Aivis 一覧を更新** — インストール済みモデル一覧を再取得

選択した音声モデルとスタイルは次回起動時に復元される。

**その他**

- **透過（ESCキー）** — ウィンドウ透過のオン / オフ
- **サポートページ** — GitHub のリリースページを開く
- **ヘルプ** — 基本操作とキーバインドを表示
- **初期設定に戻す** — モデル・Hub ログイン・音声・カメラなどをリセット
- **アプリを終了 (Quit)** — アプリを終了

### ダンス

会話中に、ダンス名を話しかけるとキャラクターが踊ります。

| 例                     | ダンス         |
| ---------------------- | -------------- |
| 「コンコンダンスして」 | コンコンダンス |
| 「かわいいダンスして」 | かわいい       |
| 「ダンスやめて」       | 停止           |

ダンス中にキャラクターをタッチすると停止できます。ダンス中もマウスで体の向きを調整できます。

### キャラクター操作

| 操作                   | 動作                                             |
| ---------------------- | ------------------------------------------------ |
| マウス移動             | 体・頭・目がカーソル方向を追う                   |
| `F` キー               | 視線追従を固定 / 解除                            |
| `P` キー               | ポーズ撮影モードの開始 / 終了                    |
| `M` キー               | マイク送信のミュート切替                         |
| ドラッグ               | ウィンドウを移動                                 |
| クリック（頭・胸など） | タッチリアクション（表情・音声・アニメーション） |

会話中に AI がポーズモードを開始することもある。ポーズ中はマウスで体の向きを調整できる。

### カメラ操作

| 操作              | 動作                          |
| ----------------- | ----------------------------- |
| `↑` / `↓`         | カメラの高さを下げる / 上げる |
| マウスホイール    | ズーム（上=寄る、下=離れる）  |
| `Cmd` + `↑` / `↓` | ズーム（上=寄る、下=離れる）  |

カメラの高さとズームは次回起動時に復元される。「初期設定に戻す」で規定値に戻せる。
