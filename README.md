# VrmAiFriend

VRM キャラクターと Gemini Live でリアルタイム音声会話ができるデスクトップコンパニオンです。

## 必要な設定

起動前に、以下を用意してください。


| 設定            | 場所     | 内容                 |
| --------------- | -------- | -------------------- |
| Gemini API キー | `~/.env` | `GEMINI_API_KEY=...` |


## 使い方

1. アプリを起動する。
2. マイクに向かって話す。起動時に Gemini Live へ自動接続される。
3. AI の応答が音声で再生され、口パク（リップシンク）が同期する。

### メニュー操作

`ESC` キーでメニューとウィンドウの透明表示を切り替えられる。

- **ハートしたモデルを選択** — 利用可能なハートしたモデルを読み込む
- **声色設定** — 性別・プリセットボイス・ピッチを選んで「適用」
- **初期設定に戻す** — モデル・声色・Hub ログイン状態をリセット
- **終了** — アプリを終了



### キャラクター操作


| 操作       | 動作                                                            |
| ---------- | --------------------------------------------------------------- |
| マウス移動 | 体・頭・目がカーソル方向を追う                                  |
| `F` キー   | その時点のマウス位置で視線追従を**固定**（もう一度 `F` で解除） |
| ドラッグ   | ウィンドウを移動f                                               |
