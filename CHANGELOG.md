# Changelog

## [0.7.0] — 会話の表情と好感度システム

**リリース名:** 会話の表情と好感度システム

### Added

- **会話連動の VRM 表情** — 返答の感情に合わせて表情を変化（happy / sad / angry / surprised / relaxed / neutral）
- **表情のクロスフェード** — 表情切り替えを滑らかに表示
- **親密度レーティング（好感度）** — 0〜100 で管理。会話の感情確定時に自動で加減算し、口調・態度に反映
- **感情タブ** — 現在の表情・好感度メーターの表示、好感度のリセット
- **設定タブ** — Gemini API キーのアプリ内保存（暗号化）、Gemini Live・VRoid Hub・Aivis の接続状態表示
- **記憶を見る** — キャラタブから覚えている内容をダイアログで確認
- **ダンス 4 種** — 超絶かわいい、コンコンダンス、キャラメルダンス、ウマウマパラパラ
- **会話での即時停止** — 「ストップ」「やめて」などでダンス・ポーズを止められる
- 好感度の段階が変わったとき、AI の口調ガイダンスが会話に反映される

### Changed

- メニューを再構成（**設定 / モデル / キャラ / 音声 / 話題 / 感情 / カメラ / About**）
- 「人格」タブを **キャラ**、「AI」タブを **話題** に改称
- 人格設定の **保存** 時にシステムプロンプトを自動生成してから保存・再接続する
- 人格の性格パラメータを **1〜5 段階**（5 段階ラベル付き）に変更（旧 0〜10 から移行）
- キャラ設定・記憶・好感度・音声・話題・カメラ・モデルを **タブごとに個別リセット**（About の一括リセットを廃止）
- 「会話の記憶を削除」を **記憶を削除** に改称し、**記憶を見る** を追加
- ダンス名の別名マッチングを整理（超絶かわいい、キャラメル、ウマウマ など）
- Gemini API キーを `~/.env` だけでなく **設定タブからも保存** 可能に

### Fixed

- AI 主導トークが連続で二重発話する問題
- Play 停止時に表情コントローラのフェードが動かない問題
- タッチ音声の一部ファイル名が読み込めない問題（濁点付き日本語ファイル名）

## [0.6.0] — 長期記憶と人格設定のアプリ内統合

**リリース名:** 長期記憶と人格設定のアプリ内統合

### Added

- **長期記憶** — 会話からユーザーのプロフィール・事実を覚え、次回以降の会話に反映
- 会話が途切れてから **10 分**後に記憶を自動整理
- メニュー **記憶** タブ（記憶の状態確認・記憶の消去）
- **人格** タブ — 基本情報・性格パラメータ・システムプロンプトの編集・自動生成・保存（従来の VrmAiFriend Config 相当をアプリ内に統合）
- メニュー **カメラ** タブ（操作説明・現在値・リセット）
- `Option` + `W`/`A`/`S`/`D`/`E`/`Q`/`R`/`F` でカメラの高さ・チルト・左右・ズームを微調整（次回起動時に復元）
- 音声タブの **Aivisを開く** ボタン
- 音声タブの **マイクレベルメータ**（調整済みなら閾値の縦線表示）
- **マイク感度調整** — スピーカー利用時のエコー対策（未調整時は起動後に自動で 1 回実施）
- Aivis 再生中の **エコーゲート** — 弱い音（エコー）を遮断し、強い発話で割り込み
- 音声の **完全全二重化** — AI 発話中もマイク送信・割り込みが可能
- 時刻の質問に日本時間（JST）で答える

### Changed

- メニューをタブ構成に整理（モデル / AI / 人格 / 記憶 / 音声 / カメラ / About）
- メニュー左上の折りたたみボタン、各タブの接続状態表示、使いやすさの改善
- 「VRoid Hub」タブを「モデル」に、「このソフト」を「About」に変更
- カメラのチルト・左右角も次回起動時に復元
- マイクミュートのショートカットを `M` から **Option + スペース** に変更
- **人格** タブ — Gemini 未接続時も保存済み設定を表示
- 文字起こしログを入力 / 出力で分離（Inspector で個別に有効化）
- 発話トリガー再接続と割り込み発話の認識を改善
- ダンス・ポーズ中止処理の共通化

**リリース名:** 会話への割り込み対応と AI 主導トーク導入

## [0.4.0] — ダンス対応・カメラ調整

**リリース名:** ダンス対応・カメラ調整

### Added

- AI 会話からのダンス再生（Function Calling: `play_dance` / `stop_dance`）
- ダンス（かわいい、コンコン など）
- カメラの高さ・ズーム調整（`↑` / `↓`、マウスホイール、`Cmd` + `↑` / `↓`）

### Changed

- ダンス・ポーズ再生中はマウス追従の感度を拡大
- タッチでダンスを停止可能（停止後のタッチリアクションは Inspector で設定）
- 「初期設定に戻す」でカメラの高さ・ズームも規定値にリセット
- 「サポートページ」をリンク表示から通常ボタンに変更
- 「ハートしたモデルを選択」を「ハートしたモデルを表示」に変更

### Fixed

- ダンス・ポーズ時のスワイプイン表示不具合
- ダンス中の水平位置ドリフトを徐々に補正

## [0.2.0] — Aivis 音声合成対応

### Added

- Aivis Speech Engine 連携（ローカル HTTP API で音声合成）
- メニューから音声モデル（サムネイル付き）・スタイルを選択
- 音声モデル / スタイル選択の永続化
- Aivis Speech 接続ステータス表示（接続 OK、起動してください など）
- Aivis 未起動時の案内音声

### Changed

- AI 応答の再生を Gemini 音声から **Aivis Speech 合成音声** に変更
- Gemini Live は文字起こし（`outputAudioTranscription`）を受け取り、Aivis で TTS
- 既定の音声モデルを「コハク / ノーマル」に設定

### Removed

- メニューの Gemini プリセット声色（性別・ボイス・ピッチ）設定

### Fixed

- 音声モデル切替が反映されない問題（`AivisSpeechPlayer` の参照統一）
- タッチ音声ファイル名の Unicode 正規化による読み込み失敗

## [0.1.0] — 初回リリース（Gemini Live API 音声合成版）

**リリース名:** Gemini Live API で音声合成版

### Added

- Gemini Live API によるリアルタイム音声会話
- Gemini 音声のそのまま再生とリップシンク
- メニューから声色（女性 / 男性プリセット）・ピッチ変更
- VRoid Hub 連携（ハートしたモデルの取得・切り替え・クレジット表記）
- 透明ウィンドウ（`ESC` でメニュー表示と切替）
- マウス追従（体・頭・目）、`F` キーで視線固定
- 自動まばたき・待機モーション
- タッチリアクション（部位クリックで表情・短い音声）

### 必要な環境

- `~/.env` の `GEMINI_API_KEY`（Aivis Speech は不要）
