# Changelog

## [0.5.0] — 会話への割り込み対応と AI 主導トーク導入

### Added

- 会話への割り込み対応（「やめて」「違う指示」などを再生中でも反映）
- AI 主導トーク（Proactive AI）のメニュー機能（無会話時間・話題・追加トピック・保存）
- Gemini Live のモデル名 / 接続状態を AI メニュー内に表示

### Changed

- Gemini Live のモデル名 / 接続状態をイベント駆動で更新（ポーリング不要）

---

## [0.4.0] — ダンス対応・カメラ調整

**リリース名:** ダンス対応・カメラ調整

### Added

- AI 会話からのダンス再生（Function Calling: `play_dance` / `stop_dance`）
- ダンス（かわいい、コンコン など）
- カメラの高さ・ズーム調整（`↑` / `↓`、マウスホイール、`Cmd` + `↑` / `↓`）

### Changed

- ダンス・ポーズ再生中はマウス追従の感度を拡大
- タッチでダンスを停止可能（停止後のタッチリアクションは Inspector で設定）
- 「初期設定に戻す」でカメラの高さ・ズームも規定値にリセット
- 「サポートページ」をリンク表示から通常ボタンに変更
- 「ハートしたモデルを選択」を「ハートしたモデルを表示」に変更

### Fixed

- ダンス・ポーズ時のスワイプイン表示不具合
- ダンス中の水平位置ドリフトを徐々に補正

---

## [0.3.0] — アニメーション・ポーズ・タッチ拡張

**リリース名:** アニメーション・ポーズ・タッチ拡張

### Added

- 待機・タッチ・ポーズ用アニメーションを大量追加
- タッチ時のアニメーション再生（頭・胸など）
- ポーズ撮影モード（`P` キーで開始 / 終了）
- AI 会話からポーズモードを開始（Function Calling）
- マイク送信のミュート（`M` キー）
- 待機モーションの連続再生オプション（Inspector で設定）

### Changed

- 起動時の Animator 初期化を `IdleMotion` に一本化
- ポーズ中はマウスで体の向きを調整可能

### Fixed

- 連続タッチ対応
- タッチ終了時の表示不具合

---

## [0.2.0] — Aivis 音声合成対応

### Added

- Aivis Speech Engine 連携（ローカル HTTP API で音声合成）
- メニューから音声モデル（サムネイル付き）・スタイルを選択
- 音声モデル / スタイル選択の永続化
- Aivis Speech 接続ステータス表示（接続 OK、起動してください など）
- Aivis 未起動時の案内音声

### Changed

- AI 応答の再生を Gemini 音声から **Aivis Speech 合成音声** に変更
- Gemini Live は文字起こし（`outputAudioTranscription`）を受け取り、Aivis で TTS
- 既定の音声モデルを「コハク / ノーマル」に設定

### Removed

- メニューの Gemini プリセット声色（性別・ボイス・ピッチ）設定

### Fixed

- 音声モデル切替が反映されない問題（`AivisSpeechPlayer` の参照統一）
- タッチ音声ファイル名の Unicode 正規化による読み込み失敗

---

## [0.1.0] — 初回リリース（Gemini Live API 音声合成版）

**リリース名:** Gemini Live API で音声合成版

### Added

- Gemini Live API によるリアルタイム音声会話
- Gemini 音声のそのまま再生とリップシンク
- メニューから声色（女性 / 男性プリセット）・ピッチ変更
- VRoid Hub 連携（ハートしたモデルの取得・切り替え・クレジット表記）
- 透明ウィンドウ（`ESC` でメニュー表示と切替）
- マウス追従（体・頭・目）、`F` キーで視線固定
- 自動まばたき・待機モーション
- タッチリアクション（部位クリックで表情・短い音声）

### 必要な環境

- `~/.env` の `GEMINI_API_KEY`（Aivis Speech は不要）
