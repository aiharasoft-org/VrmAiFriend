# Changelog

## [0.9.0] — 日常の声かけとカメラ操作の拡充

**リリース名:** 日常の声かけとカメラ操作の拡充  
**対象プラットフォーム:** Windows、macOS

### Added

- **キャラセット** — 見た目・声・性格・記憶をまとめて保存・切替
- **会話モードに集中を追加** — 雑談・作業応援・お悩み相談・音楽鑑賞・集中。能動トークの有無はモードが決める
- **タイマー／アラーム** — 「3分後に」「明日の15時に」など。複数件の確認・削除。知らせたあと返事がなければもう一度だけ知らせる
- **アプリ起動** — 「ブラウザ開いて」「メモ帳」「電卓」など、許可されたアプリの起動
- **バッテリー** — PC・マウス・キーボードの残量を聞ける。少なくなると声で知らせる
- **メディア操作** — 再生・停止・次曲／前曲など。macOS ではシャッフルや再生中の曲名も。声の音量と音楽の音量を分けて指定できる。macOS で「再生して」と頼むと、Music / Spotify が未起動なら Music を起動してから再生を始める
- **マウストラックの ON/OFF** — カメラタブから切替（既定 ON。次回起動時も保持）。頭の上下は下向きのみ
- **カメラの左右オービット** — `Option`/`Alt` + `Z`/`C` でモデル中心に右／左へ回る
- **環境音で声をかける** — 作業応援モード時のみ設定可能。大きな音などの気配から「大丈夫？」などと短く声をかけることがある（既定 OFF。音の大きさだけをアプリ内で見る）
- **会話での終了** — 「バイバイ」「さようなら」やメニューの **終了** でキャラが確認し、手を振ってから終了。会話は記憶用に保存される
- **ダンス／ポーズ中の割り込み** — 話しかけるとダンス・ポーズを止めて会話を優先
- 起動時の短い挨拶と、準備に問題があるときの音声案内

### Changed

- メニュー UI をより見やすく整理
- 会話タブのモード名を **雑談** に統一
- 会話タブの声かけ間隔を「会話が途切れてから、声をかけるまでの間隔」として分かりやすく整理
- カメラ操作ガイド（図）を現行のキー配置に合わせて更新（OS 別）
- `Option`/`Alt` + `A`/`D` は左右シフト（平行移動）、`Z`/`C` はオービット（回転）に役割を分離
- ポーズ／ダンス中のマウス追従・カメラ操作を、通常時と同じ動作に統一
- カメラタブから「現在の値」表示を削除し、マウストラックをタブ先頭へ配置
- 設定タブの **声の音量** スライダー（0〜100）

## [0.8.0] — Windows 対応版

**リリース名:** Windows 対応版  
**対象プラットフォーム:** Windows、macOS（Windows 版を新規追加。macOS は従来どおり利用可能）

### Added

- **Windows 版ビルド** — Windows / macOS の両方で動作（Aivis Speech のインストール検出・起動、接続状態表示、透過ウィンドウ）
- **カメラガイドの OS 別表示** — Windows では Alt / Ctrl、macOS では Option / Cmd のガイド画像
- **音声タブの話速スライダー** — 0.8〜1.5（既定 1.0）。次回起動時も保持
- **設定タブのマイク状態表示** — GEMINI / VRoid Hub / **マイク** / Aivis Speech の順で接続状態を表示（Windows / macOS 共通）
- **物理マイク判定** — VB-Audio 等の仮想・ソフトウェアマイクを除外し、物理マイクが無い場合はエラー表示
- Aivis 合成時の **話速調整** と先読み合成の継続
- README に **Windows CPU モード向けパフォーマンス案内**（電源モード「最適なパフォーマンス」）
- README に **Windows / macOS 別インストール手順**（SmartScreen・ブロック解除の案内を含む）

### Changed

- **マイクレベルメータ** — 実測ピーク・閾値・エコーからフルスケールを動的決定（Windows 低感度マイク向け）
- **マイク入力の安定化** — 48kHz / 44.1kHz で取得し 16kHz へリサンプル。専用 AudioSource で録音位置を進める
- **マイク感度の自動更新** — 周囲ノイズ・再生エコーを観測して閾値を静かに再調整（物理マイク利用時）
- **マイク感度調整** — 閾値計算を相対マージンに変更。調整中の赤線プレビュー改善
- **マイクの自動再試行** — 開始失敗時やセッション中の停止から定期再試行。物理マイクの接続・復帰を検知
- **物理マイクの優先選択** — USB・内蔵マイク等を優先し、仮想オーディオは入力デバイスから除外
- **ヘルプ / カメラ操作** — 実行 OS に応じて修飾キー表記を切替（Alt・Ctrl / Option・Cmd）
- カメラの矢印ズーム修飾キーを Windows では **Ctrl** に対応
- 起動時メニュータブの既定を **設定** に変更（About タブのレイアウト調整を含む）

### Fixed

- Windows で Aivis Speech が「未インストール」「未対応」と誤表示される問題
- Windows でマイクレベルメータが満タンになりにくい問題
- Windows でマイク感度調整後に閾値（赤線）が適切な位置に表示されない問題
- 仮想オーディオ（VB-Audio Virtual Cable 等）のみの環境で物理マイクありと誤判定される問題
- マイク開始直後に録音位置が進まず音声が取れない問題（Windows / macOS）
- Gemini 応答計測ログで `turnComplete` 直後の経過時間が異常に大きくなる問題

## [0.7.0] — 会話の表情と好感度システム

**リリース名:** 会話の表情と好感度システム

### Added

- **会話連動の VRM 表情** — 返答の感情に合わせて表情を変化（happy / sad / angry / surprised / relaxed / neutral）
- **表情のクロスフェード** — 表情切り替えを滑らかに表示
- **親密度レーティング（好感度）** — 0〜100 で管理。会話の感情確定時に自動で加減算し、口調・態度に反映
- **感情タブ** — 現在の表情・好感度メーターの表示、好感度のリセット
- **設定タブ** — Gemini API キーのアプリ内保存（暗号化）、Gemini Live・VRoid Hub・Aivis の接続状態表示
- **記憶を見る** — キャラタブから覚えている内容をダイアログで確認
- **ダンス 4 種** — 超絶かわいい、コンコンダンス、キャラメルダンス、ウマウマパラパラ
- **会話での即時停止** — 「ストップ」「やめて」などでダンス・ポーズを止められる
- 好感度の段階が変わったとき、AI の口調ガイダンスが会話に反映される

### Changed

- メニューを再構成（**設定 / モデル / キャラ / 音声 / 話題 / 感情 / カメラ / About**）
- 「人格」タブを **キャラ**、「AI」タブを **話題** に改称
- 人格設定の **保存** 時にシステムプロンプトを自動生成してから保存・再接続する
- 人格の性格パラメータを **1〜5 段階**（5 段階ラベル付き）に変更（旧 0〜10 から移行）
- キャラ設定・記憶・好感度・音声・話題・カメラ・モデルを **タブごとに個別リセット**（About の一括リセットを廃止）
- 「会話の記憶を削除」を **記憶を削除** に改称し、**記憶を見る** を追加
- ダンス名の別名マッチングを整理（超絶かわいい、キャラメル、ウマウマ など）
- Gemini API キーを **設定タブから保存** 可能に

### Fixed

- AI 主導トークが連続で二重発話する問題
- Play 停止時に表情コントローラのフェードが動かない問題
- タッチ音声の一部ファイル名が読み込めない問題（濁点付き日本語ファイル名）

## [0.6.0] — 長期記憶と人格設定のアプリ内統合

**リリース名:** 長期記憶と人格設定のアプリ内統合

### Added

- **長期記憶** — 会話からユーザーのプロフィール・事実を覚え、次回以降の会話に反映
- 会話が途切れてから **10 分**後に記憶を自動整理
- メニュー **記憶** タブ（記憶の状態確認・記憶の消去）
- **人格** タブ — 基本情報・性格パラメータ・システムプロンプトの編集・自動生成・保存（従来の VrmAiFriend Config 相当をアプリ内に統合）
- メニュー **カメラ** タブ（操作説明・現在値・リセット）
- `Option` + `W`/`A`/`S`/`D`/`E`/`Q`/`R`/`F` でカメラの高さ・チルト・左右・ズームを微調整（次回起動時に復元）
- 音声タブの **Aivisを開く** ボタン
- 音声タブの **マイクレベルメータ**（調整済みなら閾値の縦線表示）
- **マイク感度調整** — スピーカー利用時のエコー対策（未調整時は起動後に自動で 1 回実施）
- Aivis 再生中の **エコーゲート** — 弱い音（エコー）を遮断し、強い発話で割り込み
- 音声の **完全全二重化** — AI 発話中もマイク送信・割り込みが可能
- 時刻の質問に日本時間（JST）で答える

### Changed

- メニューをタブ構成に整理（モデル / AI / 人格 / 記憶 / 音声 / カメラ / About）
- メニュー左上の折りたたみボタン、各タブの接続状態表示、使いやすさの改善
- 「VRoid Hub」タブを「モデル」に、「このソフト」を「About」に変更
- カメラのチルト・左右角も次回起動時に復元
- マイクミュートのショートカットを `M` から **Option + スペース** に変更
- **人格** タブ — Gemini 未接続時も保存済み設定を表示
- 文字起こしログを入力 / 出力で分離（Inspector で個別に有効化）
- 発話トリガー再接続と割り込み発話の認識を改善
- ダンス・ポーズ中止処理の共通化
