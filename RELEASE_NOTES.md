# v0.6.0 — 長期記憶と人格設定のアプリ内統合

**リリース名:** 長期記憶と人格設定のアプリ内統合

会話の内容からユーザーのことを覚え、次の会話に活かす **長期記憶** を追加しました。あわせて、キャラクター設定とシステムプロンプトの編集を **メニューの「人格」タブ** に統合しました。

## ハイライト

- **長期記憶** — 会話からプロフィール・事実を覚え、以降の会話に反映
- **人格タブ** — 基本情報・性格・システムプロンプトをアプリ内で編集・自動生成・保存
- **記憶タブ** — 記憶の状態確認と消去
- メニュー UI の整理（タブ構成・折りたたみ・カメラタブなど）

## 長期記憶の使い方

1. いつもどおり会話する（Gemini API キーが必要）。
2. 会話が途切れてから **約 10 分** 経過すると、記憶が自動で整理・保存される。
3. 名前・趣味などが覚えられ、次回以降の会話で AI が参照する。
4. メニューの **記憶** タブで記憶の状態を確認できる。
5. 記憶を消したいときは **記憶を消去** を押す（確認後に削除）。

## 人格タブの使い方

1. メニューの **人格** タブを開く。
2. あなたの名前・AI の名前・性別・年齢・その他指示、性格パラメータ（0〜10）を編集する。
3. **自動生成** でシステムプロンプトを生成する。
4. 内容を確認し **保存** する（設定が会話に反映される）。
5. **初期値** でフォームをリセットできる（保存するまで確定しない）。

従来の別ツール [VrmAiFriend Config](../VrmAiFriend_Config/) と同等の設定を、アプリ内で行えます。

## メニュー（v0.6 での主な変更）

`ESC` でメニュー表示。左上の **▲** でメニューを折りたためる。

| タブ | 内容 |
| ---- | ---- |
| **モデル** | VRoid Hub からハートしたモデルを取得・切り替え |
| **AI** | AI 主導トークの設定、Gemini Live の状態 |
| **人格** | キャラクター設定・システムプロンプト |
| **記憶** | 長期記憶の状態・記憶の消去 |
| **音声** | Aivis Speech の接続・モデル選択・**Aivisを開く** |
| **カメラ** | カメラ操作の説明・現在値・リセット |
| **About** | バージョン・ヘルプ・初期設定に戻す |

表示レイアウトや接続状態の見やすさも改良しています。

## カメラ操作の追加

メニューの **カメラ** タブに操作一覧があります。`Option` キーを押しながら次のキーで微調整できます（次回起動時に復元）。

| キー | 動作 |
| ---- | ---- |
| `Option` + `E` / `Q` | 高さ（上昇 / 下降） |
| `Option` + `R` / `F` | チルト（上を向く / 下を向く） |
| `Option` + `A` / `D` | 左右の向き |
| `Option` + `W` / `S` | ズーム（前進 / 後退） |

従来どおり `↑`/`↓`、マウスホイール、`Cmd`+`↑`/`↓` も利用できます。

## 必要な環境

| 項目 | 内容 |
| ---- | ---- |
| Aivis Speech Engine | 音声合成に必須 |
| Gemini API キー | `~/.env` の `GEMINI_API_KEY`（会話・記憶・人格設定で使用） |

## アップグレード時の注意

- v0.5.0 からの追加要件はありません（Aivis + Gemini API キーは従来どおり）
- 長期記憶は初回から自動で有効です
- 人格設定はメニュー **人格** タブから編集してください（VrmAiFriend Config は v0.6 では更新していません）

# v0.4.0 — ダンス対応・カメラ調整

**リリース名:** ダンス対応・カメラ調整

会話からキャラクターにダンスを踊らせられるようになりました。カメラの高さ・ズームをキーボードやマウスホイールで調整でき、設定は次回起動時に復元されます。

## ハイライト

- **ダンス対応** — AI 会話からダンス再生（Function Calling）
- ダンス（コンコンダンス、かわいい など）
- ダンス名のあいまいマッチング（「きゃらめるだんすを踊って」など）
- アニメーションと音楽の同時再生
- タッチでダンス停止
- **カメラ調整** — 高さ・ズームをキーボード / マウスホイールで変更
- カメラ設定の永続化（次回起動時に復元）
- 「初期設定に戻す」でカメラも規定値にリセット
- メニューにヘルプボタンを追加

## ダンスの使い方

会話中に、例えば次のように話しかけてください。

- 「コンコンダンスして」
- 「かわいいダンスして」
- 「ダンスやめて」（停止）

ダンス中にキャラクターをタッチすると、ダンスを止められます（設定によりタッチリアクションも再生）。

## 操作の追加・変更

| キー              | 動作                                        |
| ----------------- | ------------------------------------------- |
| `↑` / `↓`         | カメラの高さを下げる / 上げる               |
| `Cmd` + `↑` / `↓` | ズーム（上=寄る、下=離れる）                |
| `P`               | ポーズ撮影モードの開始 / 終了（従来どおり） |
| `M`               | マイク送信のミュート切替（従来どおり）      |
| `F`               | 視線固定（従来どおり）                      |
| `ESC`             | メニュー表示（従来どおり）                  |

| マウス   | 動作                         |
| -------- | ---------------------------- |
| ホイール | ズーム（上=寄る、下=離れる） |

カメラの高さとズームは次回起動時に復元される。

## 必要な環境

| 項目                | 内容                                                                                                           |
| ------------------- | -------------------------------------------------------------------------------------------------------------- |
| Aivis Speech Engine | 必須。[AivisSpeech Engine](https://github.com/Aivis-Project/AivisSpeech-Engine) をインストールし、起動しておく |
| Gemini API キー     | `~/.env` の `GEMINI_API_KEY` が必要（会話本体は Gemini Live）                                                  |

## アップグレード時の注意

- v0.3.0 からの追加要件はありません（Aivis + Gemini API キーは従来どおり）
- ダンスは会話（Gemini Live）経由で開始します。マイクがミュートのときは話しかけられません

## セットアップ

1. Aivis Speech Engine をインストール・起動する
2. `~/.env` に `GEMINI_API_KEY` を設定する
3. アプリを起動する

詳しい使い方は [README.md](../VrmAiFriend/README.md)、全バージョンの変更一覧は [CHANGELOG.md](../VrmAiFriend/CHANGELOG.md) を参照してください。
