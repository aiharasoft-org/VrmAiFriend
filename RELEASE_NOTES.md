# v0.7.0 — 会話の表情と好感度システム

**リリース名:** 会話の表情と好感度システム

会話のトーンに合わせてキャラクターの **表情が滑らかに変わる** ようになりました。あわせて、会話の積み重ねで変化する **好感度（0〜100）** を導入し、AI の接し方にも反映されます。メニューは **設定 / キャラ / 感情** などに再構成され、API キーの保存や記憶の確認もアプリ内で行えます。

## ハイライト

- **会話連動の表情** — 返答の感情に合わせて VRM の表情が変化し、切り替えは滑らか
- **好感度システム** — 会話の感情に応じて 0〜100 が変動し、口調・態度に反映
- **感情タブ** — 現在の表情と好感度をメニューから確認
- **キャラタブの記憶** — 「記憶を見る」で覚えている内容を確認、「記憶を削除」で消去
- **設定タブ** — Gemini API キーをアプリ内に暗号化保存
- **人格パラメータ 5 段階化** — やさしさ・言葉遣いなど 7 項目を 1〜5 の段階で設定
- **ダンス 4 種と即時停止** — 超絶かわいいほか 3 種を追加。「やめて」でダンス・ポーズを停止

## 会話連動の表情

AI が返答するたびに、話のトーンに合った表情が適用されます。表情の切り替えはフェードで滑らかに行われます。

| 表情 | 例 |
| ---- | -- |
| happy | 嬉しい・楽しい話題 |
| sad | 悲しい・残念な話題 |
| angry | 怒り・不満 |
| surprised | 驚き |
| relaxed | 穏やか・リラックス |
| neutral | 特に強い感情がないとき |

タッチリアクション中はタッチ側の表情が優先されます。モデルによっては未対応の表情がある場合があります。

## 好感度（親密度レーティング）

好感度は **0〜100** で管理され、会話の感情が確定したタイミングで自動的に変動します。

| 範囲 | 目安 |
| ---- | ---- |
| 0〜19 | よそよそしい |
| 20〜39 | やや距離感がある |
| 40〜59 | 普通の友人 |
| 60〜79 | 親しみを込めた |
| 80〜100 | 好意を率直に表現 |

- **happy** の感情が確定すると +1、**sad** だと −1（同一ターン内の重複は抑制）
- 段階が変わると、AI の口調・態度が更新される
- **感情** タブでメーターを確認でき、**好感度を初期値に戻す** でリセットできる

## メニューの変更（v0.7）

`ESC` でメニュー表示。左上の **▲** で折りたたみ。

| タブ | 内容 |
| ---- | ---- |
| **設定** | API キー保存、Gemini Live・Hub・Aivis の接続状態 |
| **モデル** | VRoid Hub モデル、同梱モデルへの復帰 |
| **キャラ** | 名前・性格の編集・保存、記憶の確認・削除 |
| **音声** | Aivis の話者・マイク感度調整 |
| **話題** | AI 主導トークの設定 |
| **感情** | 現在の表情・好感度 |
| **カメラ** | カメラ操作・リセット |
| **About** | バージョン・ヘルプ |

About の **初期設定に戻す**（一括リセット）は廃止しました。各タブの **初期値に戻す** で項目ごとにリセットしてください。

## キャラ設定と記憶

1. **キャラ** タブで基本情報と性格パラメータ（**1〜5 段階**）を編集する。
2. **保存** を押すと設定が保存され、会話に反映される（接続中は再接続）。
3. **キャラを初期値に戻す** — 名前・性格のみリセット（**記憶は残る**）。
4. **記憶を見る** — 覚えているプロフィールや出来事を確認。
5. **記憶を削除** — 確認後に記憶を消去。

v0.6 以前の 0〜10 スケールの性格設定は、読み込み時に 1〜5 段階へ自動変換されます。

## ダンス

| 話しかけ例 | ダンス |
| ---------- | ------ |
| コンコンダンス | コンコンダンス |
| 超絶かわいい、かわいい | 超絶かわいい |
| キャラメルダンス | キャラメルダンス |
| ウマウマパラパラ | ウマウマパラパラ |
| ストップ、やめて | ダンス・ポーズの停止 |

## 操作の変更（v0.7）

v0.6 からのキーバインド変更はありません。

| キー | 動作 |
| ---- | ---- |
| `Option` + スペース | マイク送信のミュート切替 |
| `F` | 視線追従の固定 / 解除 |
| `P` | ポーズ撮影モードの開始 / 終了 |
| `ESC` | メニュー表示とウィンドウ透過の切替 |

## 必要な環境

| 項目 | 内容 |
| ---- | ---- |
| Aivis Speech Engine | 音声合成に必須 |
| Gemini API キー | **設定** タブで保存（`~/.env` からの取り込みにも対応） |

## アップグレード時の注意

- v0.6.0 からの追加要件はありません（Aivis + Gemini API キーは従来どおり）
- API キーは **設定** タブから保存できます（`~/.env` のキーは初回に取り込まれる場合があります）
- 性格パラメータは 1〜5 段階に変わりました（旧値は自動変換）
- 好感度は新規に 0 から始まります（**感情** タブからリセット可能）
- About の一括リセットはなくなりました。各タブのボタンで個別にリセットしてください
- 記憶の確認は **キャラ** タブの **記憶を見る** から行えます

# v0.6.0 — 長期記憶と人格設定のアプリ内統合

**リリース名:** 長期記憶と人格設定のアプリ内統合

会話の内容からユーザーのことを覚え、次の会話に活かす **長期記憶** を追加しました。あわせて、キャラクター設定の編集を **メニューの「人格」タブ** に統合しました。

## ハイライト

- **長期記憶** — 会話からプロフィール・事実を覚え、以降の会話に反映
- **人格タブ** — 基本情報・性格をアプリ内で編集・保存（未接続時も表示）
- **マイク感度調整とエコーゲート** — スピーカー利用時のエコー対策、AI 発話中の割り込み
- **音声全二重化** — AI が話している最中もマイク送信・割り込みが可能
- メニュー UI の整理（タブ構成・折りたたみ・カメラタブ・音声レベルメータなど）

## 長期記憶の使い方

1. いつもどおり会話する（Gemini API キーが必要）。
2. 会話が途切れてから **約 10 分** 経過すると、記憶が自動で整理・保存される。
3. 名前・趣味などが覚えられ、次回以降の会話で AI が参照する。
4. v0.7 以降は **キャラ** タブの **記憶を見る** で内容を確認できる。
5. 記憶を消したいときは **記憶を削除** を押す（確認後に削除）。

## 必要な環境

| 項目 | 内容 |
| ---- | ---- |
| Aivis Speech Engine | 音声合成に必須 |
| Gemini API キー | `~/.env` の `GEMINI_API_KEY`（v0.7 以降は設定タブからも保存可） |

## アップグレード時の注意

- v0.5.0 からの追加要件はありません（Aivis + Gemini API キーは従来どおり）
- 長期記憶は初回から自動で有効です
- スピーカー利用時は **マイク感度調整** を推奨
- マイクミュートは **Option + スペース**（以前の `M` から変更）

# v0.4.0 — ダンス対応・カメラ調整

**リリース名:** ダンス対応・カメラ調整

会話からキャラクターにダンスを踊らせられるようになりました。カメラの高さ・ズームをキーボードやマウスホイールで調整でき、設定は次回起動時に復元されます。

## ハイライト

- **ダンス対応** — AI 会話からダンス再生
- ダンス（コンコンダンス、かわいい など）
- タッチでダンス停止
- **カメラ調整** — 高さ・ズームをキーボード / マウスホイールで変更
- カメラ設定の永続化（次回起動時に復元）

## 必要な環境

| 項目                | 内容                                                                                                           |
| ------------------- | -------------------------------------------------------------------------------------------------------------- |
| Aivis Speech Engine | 必須。[AivisSpeech Engine](https://github.com/Aivis-Project/AivisSpeech-Engine) をインストールし、起動しておく |
| Gemini API キー     | 会話に必要                                                                                                      |

詳しい使い方は [README.md](README.md)、全バージョンの変更一覧は [CHANGELOG.md](CHANGELOG.md) を参照してください。
